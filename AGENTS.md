# llm-audio

Single-header C++17 library for OpenAI Whisper transcription, translation, and TTS.

## Structure
- `include/llm_audio.hpp` — single-header implementation
- `examples/` — usage examples
- `CMakeLists.txt` — cmake build (requires vcpkg curl where applicable)

## Build
```bash
cmake -B build && cmake --build build
```
